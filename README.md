# AvanteNow Weekly Reporting

Static weekly activity report dashboard for AvanteNow (18–27 March 2026). Served as a single-page HTML app.

## Deploy on Vercel

1. Import this repository in [Vercel](https://vercel.com).
2. Use the default settings (framework: Other / static; output: project root).
3. Vercel will serve `index.html` at the site root.

No build step is required.
